# naive-linux-timer-gui

A simple local visually-appealing stopwatch/timer to help me understand what kinds of accelerated 3D effects are available in a consumer Linux distribution these days.

## Features

The user can specify a time in the future or a timespan (timer), or begin a stopwatch to determine how much time has elapsed since activation (stopwatch).

The timespan in question (remaining time vs. elapsed time) is displayed in an appealing 3D-accelerated fashion (perhaps some rasterized vector font becomes a texturizing pass on a shard of glass floating in space).

When the time in the future / timespan is reached, an alarm sound is triggered, and appropriate visual effects (the shard shatters). The user may reset the timer/stopwatch, edit the parameters and reset, or start over.

Alternatively, the current time is displayed using the default visualization techniques until the application is closed.

## Operational Details
